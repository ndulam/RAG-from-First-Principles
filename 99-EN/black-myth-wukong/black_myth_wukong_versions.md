# Black Myth: Wukong

Black Myth: Wukong is a highly anticipated action adventure game developed by a Chinese game studio. Inspired by Journey to the West, it reimagines a classic myth with modern combat, cinematic presentation, and a richly detailed world.

## Game Editions

### 1. Digital Standard Edition

- Includes the base game.

### 2. Digital Deluxe Edition

- Includes the base game.
- Includes additional in-game content and a selected digital soundtrack.
- Includes the Bronzecloud Staff weapon.
- Includes the Folk Opera Mask, Folk Opera Armor, Folk Opera Bracers, and Folk Opera Buskins.
- Includes the Wind Chimes curio.

### 3. Physical Deluxe Edition

- Includes a Digital Deluxe Edition activation code and a set of physical collectibles.
- Collectibles include a steel case, a 1:1 headband replica, the Loosened Peaches scroll, a Wind Chimes necklace, a Thunderstone ring, a Golden Crow badge, stamps, postcards, and a warranty certificate.

### 4. Physical Collector's Edition

- Includes a Digital Deluxe Edition activation code and exclusive physical collectibles.
- Collectibles include a steel case, a 1:6 scale Destined One action figure, the Heart Sutra scroll, a Wind Chimes necklace, a Thunderstone ring, a Golden Crow badge, stamps, postcards, and a warranty certificate.

## Features

- Visual quality: built with a modern game engine and a strong focus on environmental detail.
- Combat: deep action systems with demanding encounters and expressive staff-based moves.
- Cultural adaptation: presents Chinese mythology and classical legends through interactive storytelling.
